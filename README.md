## How to build

This app uses PUGJS for templates. In the command prompt, run:
```cmd
npm install
npm install -g pug-cli
npm run pug
npm start
```

If you wish to package it too, run:
```cmd
mpm pack
```
**NOTE:** Run `npm dist` if you wish to package the app into a installer
